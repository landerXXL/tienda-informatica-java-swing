# 💻 Tienda Informática Java Swing

Aplicación de escritorio en **Java Swing** para gestionar clientes, pedidos y componentes de una tienda informática.

## 🚀 Tecnologías
- Java
- Swing
- Oracle JDBC (DAO)

## ✨ Funcionalidades
- Autenticación de usuarios
- Gestión de clientes
- Compra de componentes/configuraciones
- Seguimiento de pedidos e impagos

## ▶️ Ejecución
Abrir en Eclipse y ejecutar la clase `Login`.
